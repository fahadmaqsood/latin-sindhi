How to write Latin Sindhi on windows?

You will not be able to type Q, X, and V characters because Latin Sindhi does not have these characters.

So, when you press Q, two horizontal dots (◌̈) will be placed above the previous character. This can be helpful in writing characters like ä, d̈, ë, g̈, j̈ and ẗ. While SHIFT+Q will add two horizontal dots (◌̤) below the character which can be helpful in writing d̤.

The X key will add a caron (◌̌) over the previous character so that you can write alphabets with harder sounds, for ex: b̌, ç̌, ǧ, ǰ, ǩ, ň, p̌, ř and š. While SHIFT+X will add a caron (◌̬) below an alphabet so that you can write d̬.

The V key will place a breve (◌̆) over the previous character so that you can write t̆ and SHIFT+V will place an upside-down breve over the previous character so that you can write t̑.
