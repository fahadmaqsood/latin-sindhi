About Latin Sindhi

In Sindh, Pakistan the Sindhi language is written in the modified Perso-Arabic script. In India, it is written in Devanagari (Hindi) Script.

Sindhis living in Pakistan, as well as Sindhis living in India, can speak and understand each other. However, in texting, they use the English alphabets to represent Sindhi words. This can be misinterpreted most of the time because there are so many sounds in the Sindhi language which don't exist in English.

Latin Sindhi Script would allow Sindhis all over the world to communicate with each other through one common script.

Why are "we" developing it?

We are not developing it for the sole purpose of communication but programming also. Basic programming libraries written in JavaScript, Java, PHP using Latin Sindhi will make the learning of those programming languages easier for people living in Sindh. and those people who want to prevent others from understanding how their websites are coded can use libraries written in Latin Sindhi because the Sindhi language is not known by more than 75 million people.

What are some tricks to remember Latin alphabets of Sindhi Language?

  • Whenever there is a caron (ˇ) over an alphabet, it creates its harder sound. For example: b is (ب) and b̌ is (ڀ)
  • If there is an acute over an alphabet (going towards left direction), it is equivalent to two vertical dots in arabic script of Sindhi language. For example: b is (ب) and b́ is (ٻ), g is (گ) and ǵ is (ڳ)
  • If there are two horizontal dots (¨) over or below an alphabet, it is equivalent to two horizontal dots in arabic script of Sindhi language. For example: g is (گ) and g̈ is (ڱ), d is (د) and d̈ is (ڌ) and d̤ is (ڍ)
